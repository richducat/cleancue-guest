CleanCue Support

CleanCue is currently a private pilot. Include whether you use the iPhone app or Android web app, your device/browser version, and the exact step that failed.

Common fixes

  • Deck audio: use a locally stored MP3, M4A, WAV, or another format Android Chrome can decode. Bluetooth adds latency; use a wired or tested USB audio path for live evaluation.
  • Tempo Sync: both decks need a correct BPM. This matches tempo only; use the nudge controls to align beats.
  • Recording: use Record for a transition/demo clip. It stops automatically at 5:00 (or its 16 MiB safety cap); download the WebM/Opus file immediately. It is not a full-set recorder.
  • Lyrics unavailable: confirm the exact title and artist, add the album/edit if known, then try again. You can open Manual lyrics fallback and paste reviewed lyrics if no match is available.
  • Guest requests: publish the event while signed in, copy its live guest link, and keep the DJ app online. Refresh the Requests screen if a device briefly loses connection.
  • Privacy: Android users choose Library → Delete my browser data; signed-in DJs can delete synced pilot data under Events → Cloud access.
